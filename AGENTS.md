# AGENTS.md

## Objective
Build a minimal, elegant one-page portfolio site.

## Constraints
- No frameworks (React, Vue, etc.)
- No extra build steps
- Keep bundle size minimal
- Maintain clean, readable HTML structure

## Design Rules
- Core rules are contained in design-system.md

## Code Standards
- Semantic HTML only
- No inline styles
- CSS organized top-down (layout → typography → components)

## When Unsure
- Prefer simplicity over cleverness
- Prefer removing code over adding code

## Allowed Changes
- You may modify any file in /src
- Do not change project structure without explicit instruction

## Procedure
- Please refer to task.md for the next step
- Please update task.md when your last step was completed with a recommended next step
